Robot Navigation Results

These experiments were conducted using the Soft Actor-Critic (SAC) algorithm, a state-of-the-art method for reinforcement learning that balances exploration and exploitation. Each video represents a different scenario to evaluate the navigation capabilities of the robot. For more details about SAC, please refer to the original paper.

Scenario 1: Fixed Robot and Goal, No Obstacles

In this scenario, the robot and the goal are in fixed positions, and the environment is obstacle-free.

Scenario 2: Fixed Robot and Goal, With Obstacles

This scenario introduces obstacles. Both the robot and the goal remain in fixed positions.

Scenario 3: Fixed Robot, Random Goal, With Obstacles

The robot starts from a fixed position, while the goal appears in random positions within an obstacle-filled map.

Scenario 4: Random Robot and Goal, With Obstacles

Both the robot and the goal appear in random positions within an environment filled with obstacles.
